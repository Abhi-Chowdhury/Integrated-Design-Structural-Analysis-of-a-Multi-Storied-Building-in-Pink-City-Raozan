As part of the final-year B.Sc. in Civil Engineering student and CE-412 course, my team and I designed a residential building in Pink City, Raozan, Chittagong. 
1. Here, we created a floor plan, parking plan, electrical layout plan, plumbing plan, front elevation plan, and side elevation plan of a building in AutoCAD software according to BNBC 2020. 
2. Then we designed the building model, defined materials & elements, and assigned load with the help of ETABS software according to BNBC 2020. 
3. After that, we analyzed the structure, did a serviceability check and designed beam layout, column layout and foundation layout according to ETABS analysis result in AutoCAD software by following BNBC 2020. 
4. At last, we prepared the Bill of Quantity (BoQ) for the respective building.
